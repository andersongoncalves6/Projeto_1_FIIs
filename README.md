# Projeto_1_FIIs
O projeto consiste na criação de uma ferramenta simples em Excel para simular investimentos em fundos imobiliários.

A planilha permite calcular o valor total investido, o patrimônio acumulado e os dividendos mensais, ajudando o usuário a entender melhor o impacto de seus investimentos ao longo do tempo. Este modelo pode ser usado como base para futuras expansões e personalizações, oferecendo uma solução prática e acessível para investidores iniciantes.

## Alterações do projeto inicial
Além das funcionalidades construidas durante a aula, adicionei o tempo, em anos, para se chegar a 1 milhão em investimentos e também uma breve descrição das caracteríticas de cada perfil de investidor.

Conservador - Renda estável, baixo risco, previsibilidade;

Moderado - Equilíbrio entre renda e crescimento;

Agressivo -  Alto crescimento, maior risco, volatilidade.

Também criei uma logo com um nome para a planilha, AG Dividendos.
